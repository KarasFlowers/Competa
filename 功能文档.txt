系统核心能力一览：
- 多角色 Agent 协作：信息采集、分析师、报告撰写、质检四大 Agent 独立运行、协同工作
- 结构化知识抽取：定义竞品知识 Schema（功能树、定价模型、用户画像），确保输出一致性
- DAG 任务流转：Agent 间通过 function calling 结构化消息传递，支持质检打回与迭代闭环
- 信息溯源：每条分析结论标注数据来源，支持一键跳转原始链接
- 可观测性：完整的日志与 Trace 系统，每个 Agent 的决策过程可追溯
- 报告自动生成：输出标准化 SWOT 分析报告与竞品对比矩阵
端到端使用流程
1. 用户在前端界面输入竞品分析需求（目标产品、竞品列表、分析维度）
2. 系统自动分配任务至信息采集 Agent，从公开渠道抓取竞品信息
3. 采集数据经过结构化处理，符合预定义 Schema 后传递至分析师 Agent
4. 分析师 Agent 执行多维度对比分析，生成 SWOT 矩阵与功能对比表
5. 质检 Agent 对分析结果进行事实校验与完整性审查，不合格则打回重做
6. 报告撰写 Agent 将通过校验的分析结果整合为结构化竞品报告
7. 用户在前端查看完整报告，支持溯源跳转、人工修正、Agent 决策回放
